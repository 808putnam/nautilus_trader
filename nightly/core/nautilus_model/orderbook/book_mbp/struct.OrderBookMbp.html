Expand description

Provides an order book which can handle MBP (market by price, a.k.a. L2) granularity data. The book can also be specified as being ‘top only’, meaning it will only maintain the state of the top most level of the bid and ask side.

Fields§

§instrument_id: InstrumentId

The instrument ID for the order book.

§top_only: bool

If the order book will only maintain state for the top bid and ask levels.

§sequence: u64

The last event sequence number for the order book.

§ts_last: UnixNanos

The timestamp of the last event applied to the order book.

§count: u64

The current count of events applied to the order book.
